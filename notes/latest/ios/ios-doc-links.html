iOS, Swift, Xcode Doc Links

Apple Documentation

Apple Developer Documentation

The latest developer documentation. Includes API reference, articles, and sample code. https://developer.apple.com/documentation

Xcode

Xcode Help

https://help.apple.com/xcode/mac/10.2/

Xcode + Swift

Apple site, described as "Xcode combined with the Swift programming language makes developing apps easy and fun." https://developer.apple.com/swift/resources/

Swift

Swift (General)

Overview information about "Swift-The powerful programming language that is also easy to learn." https://developer.apple.com/swift/

Swift.org

https://swift.org

Swift Knowledge Base

Free example code you can take and re-use in your own projects. Swift Knowledge Base is a free, searchable collection of solutions for common iOS questions, all written for Swift 5.1. https://www.hackingwithswift.com/example-code

Glossary of Common Swift Terms

A dictionary of terms, providing definitions for all common terms used in the Swift language. https://www.hackingwithswift.com/glossary

Hacking with Swift

With more free Swift tutorials than any other site, Hacking with Swift will help you master app development with UIKit and SwiftUI. https://www.hackingwithswift.com

Swift Weekly Brief

A community-driven weekly newsletter about what’s happening in the Swift open source projects at Swift.org. Curated by Bas Broek. Started by Jesse Squires. Published for free every other Thursday. https://swiftweekly.github.io

Swift UI

SwiftUI by Example

SwiftUI by Example is the world’s largest collection of SwiftUI examples, tips, and techniques giving you over 400 pages of hands-on code to help you build apps, solve problems, and understand how SwiftUI really works. https://www.hackingwithswift.com/quick-start/swiftui
